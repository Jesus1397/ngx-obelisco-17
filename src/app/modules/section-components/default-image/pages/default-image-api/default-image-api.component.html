Referencia de API para la directiva oDefaultImage de Obelisco

La directiva oDefaultImage en Angular se utiliza para proporcionar una imagen predeterminada en caso de que la imagen original no se cargue correctamente.


Uso:

Agrega el atributo oDefaultImage a la etiqueta <img> en tu plantilla HTML. La directiva se encargará automáticamente de cambiar la imagen en caso de error de carga.


Reglas y Consideraciones:

  • La directiva oDefaultImage debe ser aplicada únicamente a elementos <img>.
  • La directiva debe estar disponible en el módulo de Angular donde se utilizará. Asegúrate de importar y declarar la directiva en el módulo correspondiente.
  • La imagen predeterminada que se mostrará en caso de error puede ser definida mediante la propiedad defaultImageUrl en la etiqueta <img>, permitiendo así la flexibilidad de usar cualquier imagen deseada.

Componentes con Directiva oDefaultImage Integrada:

Algunos componentes de Angular, como OCard, OGallery y OHighlighted, ya vienen con la directiva oDefaultImage integrada. Puedes utilizar estos componentes directamente en tu aplicación sin necesidad de agregar la directiva manualmente a las imágenes dentro de ellos.
