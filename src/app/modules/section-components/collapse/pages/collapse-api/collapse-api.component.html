Secciones básicas de Colapsable

El componente Colapsable puede ser utilizado con las propiedades mencionadas previamente. Además, Ngx-obelisco proporciona una serie de secciones predefinidas que pueden incluirse dentro de <o-collapse></o-collapse>.

Todas las secciones mencionadas tienen la propiedad customClasses: string, la cual permite aplicar clases personalizadas a los mismos.
