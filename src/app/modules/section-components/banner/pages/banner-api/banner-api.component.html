Secciones básicas de Banner

El componente Banner puede ser utilizado con las propiedades mencionadas previamente. Además, Ngx-obelisco proporciona una serie de secciones predefinidas que pueden incluirse dentro de <o-banner></o-banner>.

Tanto o-banner-text como o-banner-actions tienen la propiedad customClasses: string, la cual permite aplicar clases personalizadas a los mismos.

Ejemplo:

{{ bannerSectionsHtml }}
