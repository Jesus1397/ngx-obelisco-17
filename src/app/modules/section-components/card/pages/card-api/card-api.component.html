Secciones básicas de Tarjeta

El componente Tarjeta puede ser utilizado con las propiedades mencionadas previamente. Además, Ngx-obelisco proporciona una serie de secciones predefinidas que pueden incluirse dentro de <o-card></o-card>.

Todas las secciones mencionadas, excepto o-card-header y o-card-content, tienen la propiedad customClasses: string, la cual permite aplicar clases personalizadas a los mismos.
