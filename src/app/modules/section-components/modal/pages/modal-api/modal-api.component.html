Secciones básicas de Modal

El componente Modal puede ser utilizado con las propiedades mencionadas previamente. Además, Ngx-obelisco proporciona una serie de secciones predefinidas que pueden incluirse dentro de <o-modal></o-modal>.

Todas las secciones mencionadas tienen la propiedad customClasses: string, la cual permite aplicar clases personalizadas a los mismos.

o-modal-header tiene la propiedad isUnbordered: boolean, la cual permite establecer si no tiene borde. Por defecto, es false.
