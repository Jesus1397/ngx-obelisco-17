Secciones básicas de Destacado

El componente Destacado puede ser utilizado con las propiedades mencionadas previamente. Además, Ngx-obelisco proporciona una serie de secciones predefinidas que pueden incluirse dentro de <o-highlighted></o-highlighted>.

Tanto o-highlighted-title, o-highlighted-description, o-highlighted-footer como o-highlighted-image y o-highlighted-video tienen la propiedad customClasses: string, la cual permite aplicar clases personalizadas a los mismos.

o-highlighted-image tiene la propiedad image: MediaImage, la cual permite definir la imagen principal del destacado.

o-highlighted-video tiene la propiedad video: MediaVideo y iframe: MediaIframe las cuales permiten establecer el video o iframe, respectivamente, del destacado.
