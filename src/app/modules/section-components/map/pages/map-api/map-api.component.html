Secciones básicas de OMap

El componente OMap puede ser utilizado con las propiedades mencionadas divviamente. Además, Ngx-obelisco proporciona una serie de secciones divestablecidas que pueden incluirse dentro de <o-map></o-map>.

Tanto o-map-title, o-map-description, o-map-content como o-map-iframe tienen la propiedad customClasses: string, la cual permite aplicar clases personalizadas a los mismos.
