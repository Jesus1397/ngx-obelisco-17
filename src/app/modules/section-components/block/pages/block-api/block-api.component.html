Secciones básicas de OBlock

El componente OBlock puede ser utilizado con las propiedades mencionadas previamente. Además, Ngx-obelisco proporciona una serie de secciones preestablecidas que pueden incluirse dentro de <o-block></o-block>.

Tanto o-block-title como o-block-text tienen la propiedad customClasses: string, la cual permite aplicar clases personalizadas a los mismos.
