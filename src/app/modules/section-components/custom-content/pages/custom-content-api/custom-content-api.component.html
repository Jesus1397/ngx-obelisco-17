Referencia de API para la directiva oCustomContent de Obelisco

Directiva estructural: *oCustomContent

La directiva OCustomContent permite definir contenido personalizado de manera dinámica dentro de los componentes:


Uso:

Agrega el atributo OCustomContent a la etiqueta HTML. Por ejemplo: 

      {{customExample}}

Reglas y Consideraciones:

  • La directiva OCustomContent debe usarse en elementos que admitan contenido dinámico, como <div>, <ng-container>, etc.
  • La directiva debe estar disponible en el módulo de Angular donde se utilizará. Asegúrate de importar y declarar la directiva en el módulo correspondiente.

Componentes con Directiva OCustomContent Integrada:

  • Alerta
  • Banner
  • Bloque de trámite
  • Colapsable
  • Destacado
  • Footer
  • Header
  • Mapa
  • Mensaje de estado
  • Modal
  • Panel
  • Tarjetas
