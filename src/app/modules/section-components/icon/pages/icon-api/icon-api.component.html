Iconos de Material Icons Round

Obelisco utiliza la librería de Material Icons Rounded, donde pueden encontrarse los iconos disponibles.

Ejemplo:

<span class="material-symbols-rounded o-icon"> info </span>

Por motivos de accesibilidad, la etiqueta <span> con clase material-symbols-rounded o-icon debe tener el atributo aria-hidden="true".

Componente o-mat-icon

Selector: o-mat-icon

Presenta como propiedad customClasses: string, la cual permite aplicar clases personalizadas al mismo.

Para más información, ver Documentación de Material Icons Round.

Iconos de Boxicons

Obelisco utiliza la librería de Boxicons, donde pueden encontrarse los iconos disponibles.

Ejemplo:

<i class="bx bxs-info-circle"> </i>

Componente o-bx-icon

Selector: o-bx-icon

Presenta como propiedad customClasses: string, la cual permite aplicar clases personalizadas al mismo.

Para más información, ver Documentación de Boxicons v2.1.4.
